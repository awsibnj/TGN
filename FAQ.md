# Frequently Asked Questions (FAQ)

## General Questions

### What is WorkHub Bot?
WorkHub is a Telegram bot that connects work providers with work takers. Providers can post tasks, and workers can complete them for rewards.

### Is WorkHub Bot free to use?
Yes! The bot is free to use. You only pay for the tasks you create (as a provider) or earn from completing tasks (as a worker).

### How do I get started?
1. Start the bot: `/start`
2. Join the required channel
3. Verify your membership
4. Choose your role (Provider or Worker)
5. Complete registration
6. Start using the bot!

### Can I be both a provider and a worker?
Yes! You can register as a provider even if you started as a worker. Simply use the "Register as Provider" option in your dashboard.

## Account & Registration

### Why do I need to join a channel?
Channel membership helps us build a community and ensures users are genuine. It's a quick verification step.

### I joined the channel but verification fails. Why?
- Make sure you actually joined the channel (not just visited)
- Wait a few seconds after joining before verifying
- Check if you have the correct channel (check the link)
- Try leaving and rejoining the channel

### What information do I need to register as a provider?
- Full name
- Mobile number
- Email address
- Gender
- Date of birth
- Location
- Password
- Referral code (optional)

### What information do I need to register as a worker?
- Full name
- Mobile number
- Date of birth
- Password
- Referral code (optional)

### Can I change my registration information later?
Currently, you cannot change registration details through the bot. Contact support for critical changes.

### I forgot my password. What should I do?
Contact support through the bot's Support feature with your registered details.

## Wallet & Payments

### How do I add money to my wallet?
1. Go to Wallet → Add Balance
2. Enter amount (₹1 - ₹10,000)
3. Choose payment method (UPI or Crypto)
4. Complete payment using provided details
5. Wait 2 minutes
6. Submit UTR/Transaction ID
7. Wait for admin approval

### How long does deposit approval take?
Usually 24-48 hours. You'll be notified once approved.

### My deposit was rejected. Why?
Common reasons:
- Incorrect UTR/Transaction ID
- Payment amount doesn't match
- Duplicate submission
- Invalid payment proof

Contact support with your transaction details for clarification.

### What's the minimum withdrawal amount?
₹150

### How do I withdraw money?
1. Go to Wallet → Withdraw
2. Select method (UPI/Bank/Crypto)
3. Enter payment details
4. Enter amount
5. Submit request
6. Wait for admin processing

### How long does withdrawal take?
2-5 business days after admin approval.

### Can I cancel a withdrawal request?
Contact support immediately if you need to cancel. Once processed, withdrawals cannot be reversed.

### What are the withdrawal fees?
Check the current fee structure in the bot or contact support.

## Tasks (For Providers)

### How do I create a task?
1. Dashboard → Create New Task
2. Enter task title and description
3. Select category
4. Set reward amount
5. Specify number of workers needed
6. Add requirements/instructions
7. Submit (ensure sufficient wallet balance)

### What happens to my money when I create a task?
The total amount (reward × number of workers) is deducted from your wallet immediately and held until task completion.

### Can I cancel a task?
Contact support to cancel a task. Refunds are at admin discretion.

### How do I know when a worker submits a task?
You'll receive a Telegram notification when a worker submits proof.

### How do I review task submissions?
Currently, task submissions are reviewed by admins. Contact support for specific task reviews.

### What if a worker submits poor quality work?
Admin reviews all submissions. Poor quality work will be rejected, and the worker won't be paid.

## Tasks (For Workers)

### How do I find tasks?
Dashboard → Get New Task → Select category → Browse available tasks

### How many tasks can I accept?
No limit! Accept as many tasks as you can complete.

### What if I can't complete a task?
Contact support. Don't submit false proof as it may result in account suspension.

### What proof do I need to submit?
Follow the task requirements. Generally:
- Document: Screenshot, certificate, or report (max 300 KB)
- Video: Demonstration or proof video (max 10 MB)

You must submit at least one (document or video).

### My task submission was rejected. Why?
Common reasons:
- Didn't follow task requirements
- Poor quality proof
- Incomplete work
- Invalid files

Check the rejection reason provided by admin.

### When do I get paid?
Payment is credited to your wallet immediately after admin approves your submission.

### Can I resubmit a rejected task?
No. Each task can only be submitted once. Take another task instead.

## Referrals

### How does the referral program work?
1. Share your referral code with friends
2. They register using your code
3. You earn ₹50 bonus per valid referral
4. Bonus is credited immediately

### What's my referral code?
Find it in: Dashboard → Refer & Earn

### How many people can I refer?
Unlimited! Refer as many people as you want.

### I referred someone but didn't get the bonus. Why?
- Check if they used your code during registration
- Bonus is only for new users
- Cannot refer yourself or fake accounts
- Wait a few minutes for the system to process

## Admin & Support

### Who are the admins?
Admins manage the bot, verify transactions, and resolve disputes.

### How do I contact support?
Dashboard → Support → Describe your issue

### How long does support take to respond?
Usually 24-48 hours.

### Can I speak to a human?
Yes! Support messages go to real admins who respond via the bot.

## Technical Issues

### Bot is not responding. What should I do?
1. Wait a few minutes and try again
2. Restart the bot: /start
3. Check your internet connection
4. Check if Telegram is working
5. Contact support if issue persists

### I can't upload files. Why?
- Check file size limits:
  - Documents: 300 KB max
  - Videos: 10 MB max
- Ensure file format is supported:
  - Documents: PDF, DOCX, PPT, PNG, JPG
  - Videos: MP4, AVI, MKV
- Check your internet connection

### Buttons are not working. What should I do?
1. Try tapping again
2. Restart the bot: /start
3. Update Telegram app
4. Clear Telegram cache
5. Contact support

### Error: "Insufficient balance"
Your wallet doesn't have enough money for the operation. Add balance first.

### Error: "User not found"
Your account may not be properly registered. Contact support.

## Security & Privacy

### Is my payment information safe?
Yes. We use Supabase for secure data storage. Payment details are encrypted.

### Who can see my personal information?
Only admins can access user information for verification purposes.

### Can I delete my account?
Contact support to request account deletion.

### How is my password stored?
Passwords are hashed using SHA-256 before storage.

### What data do you collect?
- Registration information
- Transaction history
- Task activity
- Support messages

We don't share data with third parties.

## Common Errors

### "Channel membership verification failed"
- Join the channel first
- Wait before verifying
- Try leaving and rejoining

### "Invalid amount"
- Check min/max limits
- Use numbers only (no text)
- Don't include currency symbols

### "Transaction already exists"
- Don't submit duplicate requests
- Wait for pending transaction to process
- Contact support if stuck

### "File too large"
- Compress your file
- Use a different format
- Split into multiple smaller files

### "Insufficient slots"
- Task is full
- Find another task
- Check back later
